\begin{boxedabstract}
\small
Molecular dynamics (MD) simulations of elements require interatomic potentials that explain the interactions between all elements. The Modified Embedded Atom Method (MEAM) provides a framework for describing element interactions. However, published MEAM parameter sets typically cover only subsets of elements.	 When elements do not share a common MEAM library, cross-interaction terms are unavailable and molecular simulations cannot be executed. This project will try to develop an automated pipeline that will enable MEAM potential file generation using density functional theory (DFT) calculations with neural-network surrogate optimization. Quantum Espresso will be used to compute reference properties---equilibrium lattice constants, cohesive energies, bulk moduli, elastic constants, and binary formation energies---for each element and element pair. These quantities will be used to initialize MEAM parameters through analytic mapping relations derived from the Rose universal equation of state \citep{vinet1989universal}. A neural network surrogate will then be trained on LAMMPS-evaluated elastic properties (Young's modulus and Poisson's ratio) across randomized elemental compositions, enabling gradient-based back-propagation to optimize the MEAM parameter set to reproduce target elastical behavior of the compositions. The resulting pipeline will accept an arbitrary list of elements, automatically discover and merge existing MEAM libraries, fill missing cross-terms from DFT, and produce optimized multi-element potential files ready for large-scale MD simulations.

\vspace{0.5em}
\end{boxedabstract}

\introduction
In the industry, several types of steel and Al alloys are being used. The type of the alloy used are generally determined by the elastical properties of the alloys. While some mechanical parts require enough elongation some of them require stiffness. There are some widely available alloys, whose elastical properties are known. However, since all the elastical properties are generally obtained using experimental data, they are expensive and time consuming. If there were another method such as Molecular Simulations to estimate the elastical properties of any alloy, it would help creating new alloys for mechanical materials.

Molecular simulations of elements use interatomic potentials that balance computational efficiency with physical accuracy. Among common potential types, the Modified Embedded Atom Method \citep{baskes1992} extends the original EAM by including several other properties for a broad range of metallic and covalent systems. MEAM potentials have been widely used in literature for studying elastic properties, and defect energetics in metals and alloys \citep{lee2001}.

However, the challenge in multi-component MD simulation is consistent MEAM parameterizations. Published potential files typically cover small element subsets---for example, a ternary Al--Mg--Zn set\citep{Dickel_2018} or a quinary Al--Si--Cu--Mg--Fe set\citep{jelinek2012}---and the cross-interaction parameters between elements from different files are undefined. For an industrially used alloy such as Al~7075, which contains Al, Zn, Mg, Cu, and trace elements (Cr, Mn, Fe, Si, Ti), no single published MEAM file spans the full composition. Combining parameters from separate sources without proper cross-terms would lead to unphysical results.

Traditional approaches to this problem involve manual fitting of MEAM parameters against experimental or initial databases \citep{jelinek2012}. More recently, machine-learning interatomic potentials (MLIPs) trained on DFT data have shown good accuracy \citep{behler2007, shapeev2016}, but they require large training datasets and lack the readable form of MEAM. Neural-network interatomic potentials (NNIPs) can either replace or augment classical potentials \citep{mishin2021}.

This project will take a hybrid approach: rather than replacing MEAM with a purely data-driven potential, neural networks will be used as surrogate models to optimize MEAM parameters, initially generated from existing MEAM files and DFT calculations. This will preserve the computational efficiency and transferability of the MEAM formalism while using DFT accuracy for parameter initialization and neural-network optimization.

\researchquestion
Can density functional theory reference calculations with neural-network surrogate optimization produce multi-element MEAM interatomic potentials that accurately predict the elastic properties of arbitrary metallic alloy compositions, even when the constituent elements do not share a common published MEAM parameterization?

\methodology
The project will consist of five stages, each will be implemented as a modular Python component that can be used as standalone as well as a pipeline.

\textbf{Stage 1: Configuration Generation.}
Random alloy compositions will be generated from available elements and available MEAM coverages (e.g. Al, Co, Cr, Cu, Fe, Mg, Mn, Mo, Ni, Si, Ti, Zn). Each configuration will specify composition fractions and LAMMPS simulation parameters.

\textbf{Stage 2: Molecular Dynamics Evaluation.}
Each composition will be simulated in LAMMPS \citep{thompson2022} using the MEAM potential files. The elastic tensor will be computed via a strain--stress relation: uniaxial strains of $\delta = 0.001$ will be applied along three orthogonal directions and the resulting stress tensors will be used to extract the Voigt-averaged\citep{voigt} elastic constants $C_{11}$ and $C_{12}$ via central differences. Young's modulus $E$ and Poisson's ratio $\nu$ will then be computed as:
\begin{equation}
	E = \frac{(C_{11} - C_{12})(C_{11} + 2C_{12})}{C_{11} + C_{12}}, \qquad \nu = \frac{C_{12}}{C_{11} + C_{12}}
	\label{eq:elastic}
\end{equation}
Configurations yielding $\nu < 0$ will be discarded as unphysical.

\textbf{Stage 3: DFT Reference Calculations.}
For each selected element, Quantum Espresso \citep{giannozzi2009} will be used via the Atomic Simulation Environment (ASE) to compute: (i) the equation of state from 7 strained unit cells fit to a Birch--Murnaghan\citep{birch} equation, yielding the equilibrium lattice constant $a_0$, cohesive energy $E_\mathrm{coh}$, and bulk modulus $B$; (ii) elastic constants $C_{11}$ and $C_{12}$ from stress--strain calculations

\textbf{Stage 4: MEAM Parameter Initialization and Merging.}
DFT-computed properties will be mapped onto MEAM parameters using established relations: the sublimation energy $E_\mathrm{sub} \leftarrow E_\mathrm{coh}$, the equilibrium lattice parameter $a_\mathrm{lat} \leftarrow a_0$, and the Rose equation shape parameter $\alpha = \sqrt{9B\Omega / E_\mathrm{coh}}$, where $\Omega$ is the atomic volume. Cross-interaction parameters $E_c(i,j)$ will be derived from binary formation energies. Existing MEAM library files will be automatically discovered, parsed, and merged; missing parameters will be filled from DFT results.

\textbf{Stage 5: Neural Network Surrogate Optimization.}
The DFT-initialized MEAM parameters will be refined using a neural network surrogate model. Random perturbations of the parameter set will be sampled, and each sample will be evaluated in LAMMPS across all target compositions. A feedforward neural network will be trained to map MEAM parameters to predicted elastic properties ($E$, $\nu$). Gradient-based inverse optimization through the trained surrogate will optimize the parameter set that minimizes the discrepancy between predicted and LAMMPS simulated elastic properties. The optimized parameters will be validated by direct LAMMPS evaluation.

\expectedresults
The primary expected outcome is to create a framework that can be used to generate MEAM potential files with DFT calculation, available MEAM potentials and a neural network.

\begin{enumerate}
	\item \textbf{Unified MEAM potentials:} Complete library and parameter files covering all selected elements with consistent cross-interaction terms, enabling simulations of compositions within this element set.

	\item \textbf{DFT-quality initialization:} Elemental properties (lattice constants, cohesive energies, bulk moduli, elastic constants) and binary formation energies that agree with published DFT and experimental values, providing a physically valid starting point for optimization.

	\item \textbf{Neural network surrogate accuracy:} A trained surrogate model that predicts the elastic response of MEAM parameter sets with back-propagation.
\end{enumerate}
