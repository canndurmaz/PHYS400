4 Pipeline Overview ~55 s · ~140 words
That middle path is this five-stage pipeline. Stage 1 generates random alloy compositions. Stage 2 runs each composition through LAMMPS to measure its elastic properties — Young's modulus, Poisson's ratio, and the elastic constants. Stage 3 runs DFT in Quantum Espresso for every element and every binary pair, producing first-principles reference values. Stage 4 uses those DFT results to initialize a merged, multi-element MEAM library — including the missing cross-terms. (beat) Stage 5, shown dashed and planned for next semester, will train a neural-network surrogate on Stage 2's data, then inverse-optimize the MEAM parameters to hit target elastic properties. The blue dashed branch above is a separate ML sanity check — it predicts elastic properties from composition, validates the dataset, but never feeds back into the fitting loop.
