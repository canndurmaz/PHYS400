\begin{frame}{Stage 5a: Composition $\to C_{ij}$ Surrogate (ML Sanity Check)}
\begin{columns}[T]
\column{0.48\textwidth}
\textbf{Role.}
A divergent ML branch that consumes the LAMMPS elastic database
directly and never feeds back into MEAM fitting. \emph{Two purposes}:
\begin{itemize}\small
  \item Sanity-check the MD output --- if a composition-only NN
        cannot recover $(C_{11}, C_{12})$, the dataset itself is
        suspect.
  \item Deliver a fast predictor for novel compositions
        ($<\!1$\,ms per query).
\end{itemize}

\textbf{Architecture.}
$12 \to 32 \to 20 \to 2$ MLP with ReLU; input is the
12-element mass-fraction vector $\{c_i\}$, output is
$\hat C_{11}, \hat C_{12}$.

\textbf{Loss.} Huber ($\delta = 1$, scaled units) on $C_{ij}$ ---
bounds outlier gradients from Mg/Zn-rich points.

\textbf{Deep ensemble~\citep{lakshminarayanan2017}.} Five networks
trained with independent seeds; the predictor returns
mean $\pm$ std for epistemic uncertainty quantification.

\column{0.49\textwidth}
\centering
\includegraphics[width=0.96\linewidth]{figures/training_history.png}\\[-0.2em]
\scriptsize Per-epoch training history of one ensemble member.
Huber loss (top) decays monotonically; MAE on $(E,\nu)$ (bottom)
falls below the 20\% mean-validation gate, which triggers
early-stopping.
\end{columns}
\end{frame}

\column{0.36\textwidth}
\textbf{Validation on an Al\,7075-like composition.}

\smallskip
For the preset \emph{AA7075-like}\\(Al--Zn--Mg--Cu--Cr) the surrogate
returns
\begin{itemize}\small
  \item $\hat E = 75.5$\,GPa
  \item $\hat \nu = 0.334$
\end{itemize}
in line with the published Al\,7075 range
($E\!\sim\!71$--$72$\,GPa, $\nu\!\sim\!0.33$).

\smallskip
\textbf{Distribution.} Packaged as a portable AppImage so the
trained ensemble runs on any modern Linux desktop with no Python
or CUDA toolchain installed.
