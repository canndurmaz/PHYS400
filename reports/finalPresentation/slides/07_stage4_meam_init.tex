\begin{frame}{Stage 4: MEAM Single-Element Initialization}
\begin{columns}[T]
\column{0.45\textwidth}
The MEAM screening exponent $\alpha$ is recovered from each
element's DFT triple
$\bigl(a_0, E_{\mathrm{coh}}, B\bigr)$ via
\begin{equation*}
  \alpha \;=\; a_0 \,\sqrt{\frac{9\,B\,\Omega}{E_{\mathrm{coh}}}},
\end{equation*}
where $\Omega$ is the primitive cell volume.

\smallskip
This populates a single-element MEAM block per element, which
then feeds the cross-term refitter of Stage~5.

\smallskip
\textbf{Mn caveat.} The original EOS scan for $\alpha$-Mn did not
converge; values shown below are from a relaxed BCC approximation
and the corresponding cross-terms inherit the higher uncertainty.
